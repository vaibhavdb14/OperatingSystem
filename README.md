# üíª Operating System and System Programming Programs

This repository contains practical implementations and algorithms related to **Operating Systems (OS)** and **System Programming** concepts, performed as part of practical coursework.

---

## üìö Repository Structure

The repository is organized into directories, each dedicated to a specific topic or algorithm:

| Directory Name | Description | Key Concepts Included |
| :--- | :--- | :--- |
| **Assembler** | Implementation of a basic **Assembler** or related concepts like one/two-pass assembly. | Assembly Language, Instruction Format, Symbol Table. |
| **Bankers-Algo** | Implementation of the **Banker's Algorithm** for deadlock avoidance. | Deadlock Prevention, Safe/Unsafe State, Resource Allocation. |
| **DLL** | Programs related to **Dynamic Linking Loaders (DLLs)**. | Dynamic vs. Static Linking, Loading Mechanisms, Libraries. |
| **Macro** | Implementation of a **Macro Processor** or related concepts. | Macro Definition, Expansion, Argument Handling. |
| **Memory-Allocation-Algos** | Implementations of various **Memory Allocation Algorithms**. | First Fit, Best Fit, Worst Fit, Fragmentation. |
| **Page-Replacment-Algos** | Implementations of different **Page Replacement Algorithms**. | FIFO, LRU, Optimal Page Replacement, Thrashing. |
| **Reader-Writer-Prob** | Solutions for the classic **Reader-Writer Synchronization Problem**. | Concurrency, Synchronization, Semaphores/Mutexes. |
| **Schedeuling-Algos** | Implementations of various **CPU Scheduling Algorithms**. | FCFS, SJF, Priority Scheduling, Round Robin, Context Switching. |

## üõ†Ô∏è Requirements and Setup

To run and test these programs, you typically need the following:

* **Java Development Kit (JDK):** Most modern OS and System Programming simulations are often implemented in Java, C, or C++.
* **A suitable compiler/interpreter:** (e.g., `javac`/`java` for Java, `gcc` for C/C++).
* **IDE (Optional):** An Integrated Development Environment like VS Code or IntelliJ IDEA.

### How to Run:

1.  **Clone the Repository:**
    ```bash
    git clone [Your Repository URL]
    cd [Your Repository Name]
    ```
2.  **Navigate to a Program:**
    ```bash
    cd Memory-Allocation-Algos 
    ```
3.  **Compile and Run:** (Example for a Java file named `FirstFit.java`)
    ```bash
    javac FirstFit.java
    java FirstFit
    ```
